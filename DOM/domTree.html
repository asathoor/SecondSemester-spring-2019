About the Document Object Model

A) Definition

The Document Object Model (DOM) is an application programming interface (API) for valid HTML and well-formed XML documents. It defines the logical structure of documents and the way a document is accessed and manipulated.

If you understand the DOM, you know what JavaScript does. A JavaScript will not change the initial code written by the programmer. JavaScripts may change the look of things, because the script work on the DOM tree, as you see it in the browser.

Manipulation is the keyword here.
